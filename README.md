# gotping
TCP ping tool

## Installation

```
wget -O gotping https://github.com/yozel/gotping/releases/download/v0.1.1/gotping-{OS}-{ARCH}
chmod +x gotping
```

## Usage

```
NAME:
   gotping - TCP connect/close ping
USAGE:
   gotping [options] host port

OPTIONS:
   --timeout value, -t value  (default: 1)
   --sleep value, -s value    (default: 1)
   --count value, -c value    (default: 0)
   --help, -h                 show help
   --version, -v              print the version
```
